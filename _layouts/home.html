About Me

The business...

I am a full stack Data Scientist and Machine Learning Engineer with hands-on multi-cloud experience. I specialize in designing & implementing Data and Machine Learning solutions on the Cloud.

Machine Learning
Model Building, Linear/Nonlinear Regression, Prediction, Hypothesis Testing, Classification, Clustering, Deep Learning (CNN, RNN, LSTM), Natural Language Processing and Data Visualization
I hold the AWS Certified - Machine Learning Specialty certification.
Data Engineering
Relational database management, noSQL paradigms, lambda architecture, Hadoop ecosystem, Spark and data warehousing.
I hold the AWS Certified - Big Data Specialty certification.
Cloud based Data Solutions
I design, architect and build end-to-end data solutions on the cloud. I have worked on the 3 big cloud platforms: Amazon Web Services, Google Cloud Platform, and Microsoft Azure, and have delivered many Machine Learning projects on them. For more details on my projects, check out my Portfolio page.
Trainings and mentorship
I have led and delivered multiple training in-house for developing Data Science and Data Engineering skill-sets , especially on the Cloud.
Other than that, I have successfully designed and delivered formal training for the Azure Data Scientist Associate certification exam (DP-100) to classes of ~40-50 students.
